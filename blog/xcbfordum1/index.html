The Bare Basics

Every xcb program starts by declaring the xcb connection.

xcb_connection_t *dpy;
dpy = xcb_connect(NULL,NULL);
/* if the values are NULL it gets the display from the $DISPLAY env variable */

Next we need to declare the screen

xcb_screen_t *scr;
scr = xcb_setup_roots_iterator(xcb_get_setup(dpy)).data;

All of this may seem confusing, BUT, don't worry. It gets worse from here.
Now it's time to work on the main function

#include <stdio.h>
#include <xcb/xcb.h>

xcb_connection_t *dpy;
xcb_screen_t *scr;
int main(void){
  dpy = xcb_connect(NULL,NULL);
  scr = xcb_setup_roots_iterator(xcb_get_setup(dpy)).data;
